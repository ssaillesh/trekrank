This browser UI mirrors the SwiftUI iOS app and talks to the real FastAPI backend on .

Try it:

  1. Register a user.
  2. Trips → + add a trip (e.g. Toronto, CA → Tokyo, JP).
  3. Watch the Celery worker geocode & compute distance (status flips to done).
  4. Check Profile for stats + auto-earned badges, Map for the world map, Board for the global leaderboard.
  5. Generate a share card from Profile.

The native iOS app lives in ios/TrekRank/ — project generated at TrekRank.xcodeproj (open in Xcode, ⌘R for the Simulator).
